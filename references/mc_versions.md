## 版本推荐策略

1. **追求最大兼容性** → 选择 `1.20.1` 或 `1.19.4`（Mod 生态最成熟）
2. **追求最新技术** → 选择 `1.20.4`（活跃 Mod 最多）
3. **老机器优化** → 选择 `1.12.2` 或 `1.7.10`（轻量 Mod 多）
4. **测试新 Mod** → 选择 `1.20.4`（大多数新 Mod 首发平台）

---

## 版本过滤注意事项

- **模糊版本匹配**：若用户说 "1.20"，应同时返回 1.20.1、1.20.2、1.20.4 等子版本
- **快照版本**：建议提醒用户快照版本 Mod 兼容性差
- **版本降级**：若某 Mod 不支持最新版本，建议告知用户可用的次新版本
- **LTS 版本**：Minecraft 官方未设 LTS，但 Forge/Fabric 社区会维护稳定分支
